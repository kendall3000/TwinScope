Project Overview

Prep365: A Meal Preparation Application

Prep365 is a Java-based meal preparation application designed to assist users with meal planning based on their health goals. Users can log in to create personalized meal plans, track their BMI by entering height and weight, and receive recipes based on their preferences. The app provides a detailed and user-friendly interface to help individuals meet their dietary needs.

What is Prep365?

Prep365 is a comprehensive meal preparation application that enables users to log in, calculate their BMI, and receive tailored meal plans and recipes based on their inputs. It emphasizes a user-friendly experience, ensuring individuals can easily navigate their dietary needs while managing their health goals.

My Development Process

Developing Prep365 involved several key stages:

  • Initial Planning: Collaborated with the team to define core features, including the BMI calculator, login functionality, and meal planning. Mapped out user flows and identified potential challenges.
  • Design: Created wireframes for the app's user interface and mapped out the Model-View-Controller (MVC) design. Focused on delivering a smooth and intuitive user experience.
  • Implementation: Primarily used Java for backend logic, implementing the BMI calculator, login system, and meal planner. Integrated a public recipe API for meal suggestions and connected it to user inputs for personalized results.
  • Challenges: Overcame challenges like managing API requests, securing user data, and designing a flexible meal planner that accommodates various user inputs.
  • Testing: Conducted rigorous testing of the login system, BMI calculator, and meal planner. Verified that the allergy filter works accurately and the app delivers suitable meal suggestions.

Technologies Used

  • Java: Core programming language for app development.
  • Firebase: Used for user authentication and data storage.
  • API Integration: Pulled recipe data from a public website for meal plans.
  • MVC Pattern: Used to structure the app, ensuring a clean separation of logic, interface, and data.
  • Encryption: Secured user credentials with encryption techniques.
