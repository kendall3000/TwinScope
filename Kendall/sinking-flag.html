Project Overview

Sinking Flag: An Adventure Platformer

"Sinking Flag" is an engaging adventure platformer where players navigate through diverse and challenging maps to collect flags. Featuring unique mechanics across multiple environments—underwater, forest, volcano, and laboratory—the game offers a blend of traditional platform elements with distinct challenges in each level. Notably, the volcano map, the largest in the game, features extended exploration and movement mechanics using both 'A' and 'D' keys for navigation. The game’s variety ensures an immersive experience for players as they tackle rising platforms, hydropumps, and more.

What is Sinking Flag?

"Sinking Flag" is an adventure platformer designed to challenge players with different terrains and unique gameplay mechanics for each map. Players traverse through maps like underwater, forest, volcano, and laboratory, collecting flags while navigating obstacles. The distinct mechanics in each environment, such as rising platforms in the underwater level, ensure variety and keep the gameplay fresh and exciting.

My Development Process

Developing "Sinking Flag" included the following key stages:

  • Initial Planning: Defined gameplay elements such as flag collection, unique map designs, and special mechanics for each terrain.
  • Design: Utilized hand-drawn sprites to enhance the game's visual appeal and added custom particle systems and shaders to create a polished feel.
  • Implementation: Employed the Singleton pattern to manage the game manager and used Object Pooling to optimize enemy behavior. AI pathfinding and procedural environment generation were also key components of the game.
  • Challenges: Audio issues required disabling the options menu. However, the script and logic were retained for future fixes, and the project is fully functional without sound.
  • Testing: Playtested to ensure that the gameplay mechanics, including controls and platform movement, were smooth and engaging across all maps.

Code Snippets for Design and Optimization Patterns


    // Singleton Pattern Example
    public class GameManager : MonoBehaviour {
        public static GameManager Instance { get; private set; }

        void Awake() {
            if (Instance == null) {
                Instance = this;
                DontDestroyOnLoad(gameObject);
            } else {
                Destroy(gameObject);
            }
        }
    }

    // Object Pooling Example
    public class EnemyPool : MonoBehaviour {
        public static EnemyPool SharedInstance;
        public List pooledObjects;
        public GameObject objectToPool;
        public int amountToPool;

        void Awake() {
            SharedInstance = this;
            pooledObjects = new List();
            for (int i = 0; i < amountToPool; i++) {
                GameObject obj = Instantiate(objectToPool);
                obj.SetActive(false);
                pooledObjects.Add(obj);
            }
        }
    }

Key Information Summary

"Sinking Flag" blends traditional platformer mechanics with innovative elements such as rising platforms and hydropumps. Despite the removal of audio features due to technical issues, the game remains fully playable, and the code for the audio logic is available in the GitHub repository for future improvements. AI-driven enemies, procedural environment generation, and optimization patterns, such as the Singleton and Object Pooling, add depth to the game.

Frequently Asked Questions

What are the unique mechanics in each map?
Each map features different gameplay elements such as rising platforms in the underwater map, large exploration areas in the volcano map, and different challenges in the forest and laboratory maps.
How did you implement AI?
AI in "Sinking Flag" uses predefined behaviors and state machines for complex enemy actions, as well as pathfinding to navigate procedurally generated environments.
Why is there no audio in the game?
Due to technical issues with the audio implementation, it was disabled in the final submission, but the logic and scripts remain in the GitHub repository.
