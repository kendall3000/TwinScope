Project Overview

Ensure Your Travels Are Bed Bug-Free

YourBugRep is a cutting-edge travel companion app designed to empower users with vital information and tools to avoid bed bugs during their travels. Built with modern web technologies, it features a robust database of bed bug complaints powered by Firebase and Firestore, allowing for real-time data synchronization and updates. The front end is crafted using HTML, CSS, and JavaScript, ensuring a responsive and user-friendly experience across all devices. With Google Cloud Functions integrated for real-time alerts, YourBugRep delivers timely notifications to users, enabling them to make informed decisions about their accommodations.

What is YourBugRep?

YourBugRep is a comprehensive travel companion tailored to help users avoid bed bug encounters while traveling. The app provides:

  • Detailed Database: Access to a vast database of bed bug complaints, allowing users to view reports from other travelers about specific hotels and locations.
  • Essential Safety Guides: Helpful resources and tips to recognize and prevent bed bug infestations, ensuring a safe stay.
  • Real-Time Alerts: Notifications based on user location and updates from the complaints database, ensuring users are informed of any recent bed bug activity in their vicinity.
  • Customizable Report Filters: Users can filter reports by location, date, and severity to tailor their searches and make better-informed decisions about where to stay.
  • Regular Updates: Continuous updates on bed bug activity in various locations, keeping users informed and safe throughout their travels.

YourBugRep aims to transform the way travelers experience their journeys, providing peace of mind and enhancing travel enjoyment by mitigating the risk of bed bug encounters.

My Development Process

Building YourBugRep involved several stages:

  • Initial Planning: Identified the core features needed, such as the complaints database and alert system.
  • Design: Created wireframes and mockups to visualize the user interface and user experience.
  • Implementation: Used HTML, CSS, and JavaScript for the front end, while Firebase and Firestore were utilized for the backend. Integrated Google Cloud Functions for real-time alerts.
  • Challenges: Faced issues with real-time data synchronization, which were resolved through effective use of Firebase's real-time capabilities and careful management of cloud functions.
  • Testing: Conducted extensive testing to ensure the accuracy and reliability of the database and alert features.

Technologies Used

  • HTML, CSS, JavaScript
  • ReactJS
  • Firebase and Firestore
  • Google Cloud Functions

Frequently Asked Questions

How accurate is the bed bug data?
The data is sourced from user-generated reports and may not be completely accurate. We strive to provide reliable information, but users should use their discretion.
How can I manage my membership?
Membership management can be done through the Stripe dashboard. Premium accounts have access to full alert capabilities and auto-renewal features.
Can I use YourBugRep on mobile?
Yes, the application is fully responsive and can be used on mobile devices.
