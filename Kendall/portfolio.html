YourBugRep Website

YourBugRep is a comprehensive travel companion tailored to help users avoid bed bug encounters while traveling. t uses Firebase and Firestore for real-time data on bed bug complaints, providing up-to-date information. The app features a responsive interface built with HTML, CSS, and JavaScript, ensuring a smooth user experience across devices. Google Cloud Functions power real-time alerts, allowing users to receive notifications about potential bed bug risks at their accommodations.
