YourBugRep Website

YourBugRep is a comprehensive travel companion tailored to help users avoid bed bug encounters while traveling. It uses Firebase and Firestore for real-time data on bed bug complaints, providing up-to-date information. The app features a responsive interface built with HTML, CSS, and JavaScript, ensuring a smooth user experience across devices. Google Cloud Functions power real-time alerts, allowing users to receive notifications about potential bed bug risks at their accommodations.

Rasa Chatbot

The Rasa chatbot was designed to streamline customer interactions through machine learning-based dialogue management, capable of handling both common and complex queries. As an open-source framework for building contextual AI assistants, Rasa combines natural language understanding (NLU) and dialogue management to create flexible chatbots. This project specifically aimed to integrate Rasa with a custom action server for personalized user experiences.

Flappy Bird Game

The Flappy Bird Clone is a browser-based game created using JavaScript and HTML5 Canvas. Players control a bird that must navigate through pipes by tapping the screen to avoid obstacles. The game features randomized pipes, a real-time score counter, and a game-over screen when the bird collides with a pipe or flies off-screen. This project helped me strengthen my skills in front-end development, interactive web applications, and game logic.

Info / Git Repository
