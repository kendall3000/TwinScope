Project Overview

Random Quote Generator: Generate Inspirational Quotes

The Random Quote Generator is a simple web application built using Node.js for the backend and JavaScript for dynamic functionality. It fetches random quotes from an external API and displays them to users with a button click. The app includes a dark/light mode toggle, implemented with JavaScript to enhance user experience and accessibility. The frontend is designed to be responsive, ensuring compatibility across both desktop and mobile devices.

What is the Random Quote Generator?

The Random Quote Generator allows users to easily view a different inspirational quote with each click of a button. Users can also switch between light and dark mode for a better viewing experience. The app makes use of an API to fetch the quotes and display them dynamically in the browser.

My Development Process

Creating the Random Quote Generator involved the following key stages:

  • Initial Setup: Set up the basic structure of the webpage with HTML, CSS, and JavaScript.
  • API Integration: Integrated the free API to fetch random quotes and display them dynamically in the browser.
  • Dark Mode Toggle: Implemented a toggle button that switches the website's theme between light and dark mode using JavaScript.
  • UI/UX Design: Styled the page to ensure it is clean and user-friendly, with clear fonts and simple navigation.
  • Testing: Continuously tested the functionality of the app to ensure the API was fetching quotes correctly and that the theme toggled smoothly.

App Controls

  • Generate Quote Button: Click to fetch a new random quote.
  • Dark Mode Toggle: Click the button to switch between light and dark themes.

Technologies Used

  • Node.js (Backend for API calls)
  • JavaScript (Frontend logic, Dark Mode toggle)
  • CSS (Styling for responsive layout and dark mode)
  • HTML (Structure of the webpage)
  • Free Public API (For random quotes)

Code Snippets for the Random Quote Generator


// API call to fetch random quotes
fetch('https://api.quotable.io/random')
    .then(response => response.json())
    .then(data => {
        document.getElementById('quote').innerText = `"${data.content}" - ${data.author}`;
    });

// Dark Mode Toggle
const toggleButton = document.getElementById('toggle-theme');
toggleButton.addEventListener('click', () => {
    document.body.classList.toggle('dark-mode');
});

Frequently Asked Questions

What is the source of the quotes?
The quotes are fetched from a free public API, quotable.io.
Can I customize the quotes?
Currently, the app fetches quotes randomly, but customization features can be added in the future.
How does the dark mode toggle work?
The dark mode toggle button adds a "dark-mode" class to the body, switching the theme based on user preference.
