Project Overview

eBookfair: A Scholastic E-Commerce Experience

The eBookfair is a full-stack e-commerce application developed as a team project for the Software Engineering course (CS 3773) in Fall 2023. This Django-based web application, connected to a MySQL database, dynamically populates HTML templates with product information. It features basic search functionality for products by name or description and allows users to create accounts. Styled with a color scheme reminiscent of classic Scholastic Book Fairs, the application offers a nostalgic touch while delivering a modern, responsive web interface using Bootstrap.

What is eBookfair?

eBookfair is a comprehensive e-commerce platform designed to provide users with a seamless shopping experience. It includes a Django web application that interacts with a MySQL database to manage product information. Key features include a search function to find products, user account creation, and a responsive interface that adapts to various devices. My Development Process

Developing eBookfair involved several key stages:

  • Initial Planning: Defined core features such as the product database, search functionality, and user account management.
  • Design: Created wireframes and mockups to visualize the user interface and experience, ensuring alignment with the Scholastic Book Fair theme.
  • Implementation: Utilized Django for the backend and MySQL for the database, with Bootstrap for responsive design. Dealt with challenges like real-time data synchronization and Django's limitations on composite keys.
  • Challenges: Encountered issues with project management and team coordination, which affected the final product. These included incomplete features like shopping cart and checkout functionality.
  • Testing: Performed extensive testing to ensure the reliability and performance of the application despite developmental hurdles.

Technologies Used
