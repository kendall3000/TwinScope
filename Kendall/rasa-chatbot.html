Project Overview

Rasa Chatbot Implementation

The Rasa chatbot was designed and implemented to streamline customer interactions. It uses machine learning-based dialogue management to handle both common and complex queries.

Running the Chatbot

Follow these steps to run the Rasa chatbot:

  1. Open a terminal or command prompt window.
  2. Run the command: rasa run actions to start the action server.
  3. Open a new terminal or command prompt window.
  4. Run the command: rasa shell to start interacting with the chatbot.

What is Rasa Chatbot?

The Rasa Chatbot is an open-source framework for building contextual AI assistants. It combines natural language understanding (NLU) and dialogue management to create flexible and adaptable chatbots. This project aimed to integrate Rasa with a custom action server for personalized user experiences.

My Development Process

The development of the chatbot involved several key steps:

  • Initial Planning: Defined use cases and customer interaction workflows.
  • Training the NLU Model: Used Rasa's training data format to define intents, entities, and responses.
  • Custom Actions: Created custom actions for dynamic responses using Rasa SDK and Python.
  • Testing: Tested chatbot performance with sample conversations to ensure response accuracy and relevance.

Technologies Used

  • Rasa (NLU & Core)
  • Python
  • Custom Action Server
  • Command-line interface for interaction
